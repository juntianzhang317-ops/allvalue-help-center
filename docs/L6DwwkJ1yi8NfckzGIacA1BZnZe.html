微信小程序商城上线

微信小程序申请及认证

❗️ AllValue定位海外店铺经营，需要海外企业主体（包含中国香港、中国澳门、中国台湾，支持个体工商户），请提前知悉和准备；

❗️ 如果您有企业主体认证的微信公众号，并公众号的类型是服务号，可以参照微信公众号快速注册小程序，是否有开放入口，需要以微信公众后台为准。

一、微信小程序申请

开始申请

申请微信小程序，请点击跳转至微信公众平台开始申请

◽️点击”小程序“

◽️点击“前往注册“

填写账号信息并激活

每个邮箱仅能申请一个小程序；作为登录账号，请填写未被微信公众平台注册，未被微信开放平台注册，未被个人微信号绑定的邮箱

◽️输入满足以上条件的邮箱账号，设置密码，并记住您的登录邮箱和密码

◽️在收件箱接收微信小程序官方发送的激活邮件

◽️激活账号

前往您的邮箱激活您的小程序注册账号

二、微信小程序注册

填写小程序用户登记信息

注册国家/地区需属于海外，主体类型需要选择企业

填写主体信息登记

企业名称和企业注册号填写详情参考：境外热门地区/国家常见主体证件示例

进入文档后，使用control+F 或者 command + F 搜索您企业所在地的示范文件

所有需要填写的信息在文档内皆有注明

填写内容需和您提供的文件完全一致

举例：加拿大

填写管理员信息登记

填写管理员基础信息，并通过微信扫码进行绑定

管理员姓名的填写方式需要和您提供的证件一致

注意管理员手机区号，若是海外，请重选海外区号，否则收不到验证码。

三、微信小程序认证

开始前，您需要准备好以下材料：

1.《企业注册证》或者《商业许可证书》：原件照片或扫描，或者复印件加盖企业公章后扫描；保存为图片格式

2. 联系人正反面身份证件彩色扫描件； 3. 申请公函（中文简体认证公函、中文繁体认证公函、英文认证公函）； 4. 联系人手机电话对账单。若联系人手机电话对账单无法提供，可上传公司办公电话对账单（需加盖公章），或银行卡对账单； 5. 其他证明材料（若是选择商标命名的，请选择正确的商标注册书或者是商标授权书）。

如海外公司没有公章，需要加盖公章的位置需写上法人+和证件名字相同的签字

同意协议

填写企业资质信息

主体证件格式可参考：境外热门地区/国家常见主体证件示例

◽️上传《企业注册证》或《商业许可证》原件照片或扫描件

格式要求：原件照片、扫描件或者加盖公章的复印件，支持.jpg.jpeg.bmp.gif .png格式照片，大小不超过5M。

上传其他相关资料

这一步您需要准备的材料有：

◽️申请公函 境外主体账号认证公函示例

◽️运营授权书 境外主体账号运营授权书填写示例

◽️手机或银行对账单 （三个月）

◽️身份证件（正面）

◽️身份证件（反面）

*如果提供中国护照，姓名就是中文+拼音

-请注意，该步骤上传材料均不支持PDF版本，仅支持图片格式

-申请主体为公司全名

-您在运营授权书中需要填写的账号ID已展示在您的公函中 （gh_开头）

运营授权书：申请主体为公司全名。原始id查看申请公函内的id（gh_开头）。指定管理员姓名需按身份证填写（如果海外驾照是英文，就填英文）。申请主体授权代表请填写公司法人的姓名，正楷填写。海外公司无公章可以不用盖公章。

运营授权书的指引详见https://kf.qq.com/faq/220907BR7vUF220907N7Zv2q.html

填写认证联系人信息

联系人姓名填写方式需和提供的证件保持一致

如果是海外手机号码注意区号更改

支付费用

提交材料后，选择电子发票，支付认证费用

四、等待审核

如需补交资料，审核人员会在1-3个工作日内给你的预留邮箱邮件告知。按要求补交即可

邮件可以无限提交

微信后台更新信息限制最多3次

提交后如因提交的注册文件里没有显示出法人的姓名，邮件反馈或需要手持证件拍照

具体请以微信审核人员邮件反馈为准！

五、审核成功

认证成功后，您需要完成小程序基本信息和类目补充

微信小程序信息及类目完善

开始这一步之前，请确保您的【小程序发布流程】-【微信认证】为已完成状态

把小程序的信息和类目补充完成后，大概一个工作日就会审核完毕。

审核完成后就可以绑定至AllValue的后台准备开始运营您的店铺啦！

填写小程序信息

你需要登录微信公众平台（https://mp.weixin.qq.com/），在「首页」模块进入小程序发布流程页面

填写小程序发布流程中第一步的信息，如小程序名称，图标，描述等

参考https://kf.qq.com/faq/170109umMvm6170109MZNnYV.html

小程序名称如果需要更改，只能再次重新认证，请慎重填写，其他相关信息可以更改的频率可见下图。

类目选择

开始之前，请点击下方文档参考并选择合适自己的小程序类目。类目可以多选，建议您把您会涉及到的类目都勾选上；至少有2个类目通过审核，店铺即可上线

如果售卖相关商品但无法提供相应的资质，建议不选需要资质要求的类目，先通过审核将店铺上线，再上架相关商品。由于店铺已经审核通过，暂时无需被审核，在下次年审前，相关商品通常不会被审核下架。

类目选择请参考：

境外主体小程序开放的服务类目

您在选择类目时尽可能选择本地服务相关的类目

这里我们推荐选择的在本地服务下的类目；如玩具，服装 类目：

*不需要完成小程序备案

通过审核即可

微信小程序授权发布

一、授权发布的条件

1、微信小程序信息、小程序类目和微信认证都是“已完成”的状态。

如果有未完成的项目，登录微信公众平台，用微信小程序管理员扫码登录后，参照微信小程序信息及类目完善去完成；

（目前境外主体，是不需要备案的）

2.店铺装修和商品上传是初步完成的状态

登录AllValue后台，

a.按照微信小程序类目，上传部分商品，建议每个品类0-2件，总商品数不少于2件

b.极致移动已经装修，且开启「优先启用移动店铺」

二、授权发布微信小程序商城

商家登录AllValue后台，在微信小程序模块，选择“已经拥有小程序”，点击“一件生成”

用微信小程序管理微信扫描二维码，授权发布微信小程序（注意需要同意全部授权项），等待1-7个工作日，腾讯进行审核

扫码的商家微信会收到审核结果通知

注意事项

申请开通收款账户

一、确认收款账户服务商

将你申请微信小程序的公司主体所在国家/地区告知你的客户经理或者服务经理，就能够匹配能够支持开通收款账户的服务商，并协助你建联。

二、按照收款服务商的要求提供资料，并且完成开通账户的流程

具体所需资料，以收款服务商为准。如果不能提供，需要与收款服务商直接沟通和协调。

三、开户成功

获取收款服务商开通好的商户ID及支付密钥并予以保存，后期需要填入AllValue店铺后台。

PS：微信小程序开户常用的基本信息

需要到微信公众平台获取。登录网址为：https://mp.weixin.qq.com/，需要申请微信小程序的管理员微信扫码登录；

1 .小程序在微信平台上的AppID（❗️一定要告知收款服务商进行绑定，否则微信小程序中无法完成支付）

路径：「设置」>「基本设置」>「账号信息」（设置页面最底部）

2. 微信平台的注册主体名称

路径：「设置」>「基本信息」>「主体信息」

❗️ 一定要完全一致，空格或任意符号差异都有可能导致配置失败

3. 微信平台上的小程序名称

路径：「设置」>「基本信息」>「小程序名称」

收款账户配置流程（微信小程序）

一、配置收款信息

操作路径：AllValue店铺后台（https://admin.allvalue.com/admin/user#/login），「微信小程序」>「设置」>「微信小程序支付」>「选择支付提供商」

填写信息后点击激活，即可激活账号

微信小程序客服

微信小程序客服

使用场景：

消费者在小程序店铺购物，点击客服按钮可以直接发起咨询，商家通过客服微信网页端、客服小程序在48小时内和消费者进行对话，提高店铺服务质量。

功能介绍

开通小程序客服

1.在微信公众号后台登录绑定AllValue的小程序，打开小程序客服功能，至少添加一个客服人员。

请先确认小程序已经开通并授权绑定AllValue店铺

2.扫码进入小程序客服助手，设置状态为“在线”，避免错过消费者的咨询，目前小程序支持绑定不多于100个客服人员。

配置小程序客服

1.在AllValue后台-应用-微信小程序-开启小程序客服，将在小程序展示客服咨询入口。

2.小程序支持页面自定义装修，可设置是否在指定页面展示客服悬浮按钮，默认将在个人中心页、订单详情页面展示入口，支持设置客服悬浮按钮的品牌颜色。

3.小程序应用场景：店铺首页、商品详情、订单详情、商品列表、个人中心页面

客服助手小程序使用

登录并接入

已被绑定的小程序客服人员可微信搜索「客服小助手」或扫码登录「客服小助手」小程序，并选择对应的小程序账号，登录后即可看到与小程序对话的用户，可选择接入对话。

切换客服状态

点击在线状态，选择客服在线状态，即使退出客服小程序，仍可在“服务通知”中接收到用户咨询的消息提醒； 选择客服离线状态后，将无法收到客服消息与消息提醒。

接收与发送消息

打开「客服小助手」小程序后，进入“待接入列表”可选择用户会话进行接入； 已经接入的会话，客服人员可以在48小时内和用户进行对话，目前支持发送文本、图片、小程序卡片类型的消息。

网页端客服工具使用

登录并接入

已被绑定的小程序客服人员可扫码登录网页端微信小程序客服，并选择对应的小程序账号，登录后即可看到与小程序对话的用户，可选择接入对话。

切换客服状态

点击在线状态，可以选择在线状态、离线状态

接收消息

手动接入：客服人员上线后，可在“待接入”列表中，手动接入待回复的用户会话。

自动接入：当待接入的用户会话太多时，可以在设置-接入与回复中，开启自动接入。

发送消息

已经接入的会话，客服人员可以在48小时内和用户进行对话，目前支持发送文本、图片、小程序卡片类型的消息。

客服规则说明

客服消息类型：目前支持文本、图片、小程序卡片类型消息。

客服消息下发条件：小程序用户在小程序内唤起客服会话或用户给小程序客服发送消息。

当用户和小程序客服产生特定动作的交互时，小程序可向用户下发客服消息。

用户动作

允许下发条数限制

下发时限

用户发送信息

5条

48小时

可发送客服消息条数不累加，上述用户动作会触发可下发条数及可下发时限的更新，可下发消息条数更新为当前可下发条数限制的最大值，有效下发时间限制也更新为最长有效时间。

微信小程序解除授权

解除微信小程序授权

当你注册了新的微信小程序，想要用新的微信小程序和店铺绑定时，需要先将店铺和微信小程序解除绑定。或者其他需要将微信小程序和店铺解绑的情况，也可以参照如下操作：

一、在AllValue店铺后台将店铺与微信小程序解除授权

登录地址：https://admin.allvalue.com/admin/user#/login

操作路径：AllValue店铺后台「微信小程序」>「小程序基本信息」>「解除授权」

二、在微信公众平台进行确认，如果还存在绑定关系，需要彻底解除

登录地址：https://mp.weixin.qq.com/

操作路径：微信公众平台，左下角微信小程序「账号设置」>「第三方设置」>「解除授权」

需要上面两个模块全部解除授权，才能沉底解除绑定关系
