基础设置

AllValue店铺版 › 后台操作指引 › 设置 › 基础设置

在开始设置AllValue商店之前，你需要创建一个AllValue账户并为商店添加一些基本的商业信息。尽管不需要立即包含所有详细信息，不过请务必添加以下项：

一、创建AllValue账户

创建AllValue账户以开始设置你的商店。你需要输入有效的电子邮箱地址、密码、商店名称和商店域名。

你无法更改你的myallvalue域名，但你可以随时绑定你自己的域名和更改商店名称。

二、设置店铺详细信息

路径：设置-基础设置-店铺详细信息

1.店铺名称

你的商店名称是在你网站的首页上显示的名称，通常店铺的名称在SEO的时候也有重要作用，所以最好与品牌名一致。

2.店铺联系人邮箱

店铺联系人邮箱是指顾客联系你时，你接收邮件的邮箱。通常我们在系统发送的付款成功、发货通知等通知邮件中，会告知顾客，如果有疑问可以通过该邮件和你联系。

3.发件人邮箱

发件人邮箱是指系统发送通知给顾客时使用的邮箱，比如订单通知和营销活动通知等。

4.店铺行业

按照你的实际行业选择即可

三、店铺地址

你的合法公司名称和地址应与营业执照上的名称和地址保持一致，这将有助于你申请Facebook广告账户时提供相应的申请资料。

路径：设置-基础设置-店铺地址

1.法定公司名称

填写你与AllValue合作时使用的公司名称，并且按照指引完成企业认证，就可以在线下载双方合作协议。

2.店铺地址

填写你实际的经营地址即可，该地址可能出现在invoice中。

四、标准和格式

1.时区

店铺的时区将是你优惠活动生效的时间时区，建议按照您主要销售国家和地区的时区设置。

2.设置或更改商店的默认重量单位

你可以选择以下默认重量单位，选择最适合你业务的重量单位。这只是默认值，稍后可以针对各个产品进行编辑。

3.编辑订单ID格式

默认情况下，你的首页订单ID的格式为#1001。之后的每个新订单将按1递增。

你无法更改订单编号的开头部分 (1001)，但若要自定义订单 ID 的格式，可以添加或编辑订单 ID 前缀或后缀。（在订单列表检索时，只需要输入中间系统生成的数字检索，无需添加前缀和后缀）；

如果你不想设置订单编号的固定格式，想随机生成，选择“随机格式”即可，订单编号会按照日期+时分秒+随机数生成，但是随机格式订单编号不再支持更改。

五、商品货币

在AllValue后台的基础设置页面中，你可以选择店铺使用的货币（例如美元、日元、欧元、澳元等），并且可以更改货币的格式（展示方式）

1.商品币种

可以根据你的目标市场，调整你想要定价的币种

注意：店铺没有产生订单之前，币种都可以更改。当你的店铺一旦产生订单（包含测试单）货币将被锁定，无法更改。

2.货币格式

比如同样是美金定价，你可以选择展示方式是${{amount}}、USD {{amount}}、US${{amount}}这三种中的任一种

设置路径：设置-基础设置-商品货币
