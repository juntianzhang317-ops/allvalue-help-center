AllValue Link - 社交内容托管设置

allvalue link， AI托管

1. 登录账号

使用电脑打开浏览器，点击这里进入 AllValue link 登录界面，

输入账号密码后，登录到AllValue Link后台

点击【开始托管】，开始社交内容托管设置

2. 填写店铺信息

根据页面显示填写您想要提供给AI学习的商铺资料信息

3. 开始内容托管

添加需要AI智能托管的小红书账号，一个AllValue Link可以最多绑定两个小红书账号。

点击需要托管的账号，设置发布计划

选择日常内容托管，设置发布频率，发布时间，以及发布时需要用到的笔记图片

4. 设置成功

设置完成后，发布前系统会将待发布的笔记已短信形式发送链接至您的手机。

需要点击短信内链接进行笔记发布。

如何获取小红书店铺POI链接

小红书“POI”是什么？

小红书中的“POI”是指“Point of Interest”，中文可以理解为“兴趣点”或“地点标记”。在小红书上，POI通常指代具体的商家、店铺、景点、餐厅等，是用户可以进行地理位置标记和分享的地点。

简单来说，当你发布一条小红书笔记，并关联了一个具体的地点（比如一家咖啡店），这个地点就是POI。用户可以通过点击笔记中的POI，跳转到该地点的详情页，了解更多信息，比如地址、电话、营业时间、用户评价等。

什么是小红书笔记组件：

通过AI能力，润色高效辅助写商品特色和真实体验的种草笔记，顾客只需点击一键发送

什么是AllValue碰碰贴：

碰碰贴是结合NFC通讯技术的全新门店互动物料。顾客手机解锁后，无需打开APP扫码，碰一下即可快速完成各种互动，例如去小红书发1篇门店种草笔记（AI帮写又好又快）

顾客操作流程：

1、通过“碰碰贴”碰一下NFC卡片 / 扫描二维码，进入页面

2、点击行动按钮，去发小红书笔记

3、AI自动生产种草笔记：笔记标题、笔记内容、话题

4、点击【发布小红书】按钮，内容一键带入顾客的小红书主页，完成发布
