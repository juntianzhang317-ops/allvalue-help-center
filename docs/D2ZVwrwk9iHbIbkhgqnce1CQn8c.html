微信小程序申请及认证

AllValue店铺版 › AllValue微信小程序商城 › 微信小程序商城上线 › 微信小程序申请及认证

❗️ AllValue定位海外店铺经营，需要海外企业主体（包含中国香港、中国澳门、中国台湾，支持个体工商户），请提前知悉和准备；

❗️ 如果您有企业主体认证的微信公众号，并公众号的类型是服务号，可以参照微信公众号快速注册小程序，是否有开放入口，需要以微信公众后台为准。

一、微信小程序申请

开始申请

申请微信小程序，请点击跳转至微信公众平台开始申请

◽️点击”小程序“

◽️点击“前往注册“

填写账号信息并激活

每个邮箱仅能申请一个小程序；作为登录账号，请填写未被微信公众平台注册，未被微信开放平台注册，未被个人微信号绑定的邮箱

◽️输入满足以上条件的邮箱账号，设置密码，并记住您的登录邮箱和密码

◽️在收件箱接收微信小程序官方发送的激活邮件

◽️激活账号

前往您的邮箱激活您的小程序注册账号

二、微信小程序注册

填写小程序用户登记信息

注册国家/地区需属于海外，主体类型需要选择企业

填写主体信息登记

企业名称和企业注册号填写详情参考：境外热门地区/国家常见主体证件示例

进入文档后，使用control+F 或者 command + F 搜索您企业所在地的示范文件

所有需要填写的信息在文档内皆有注明

填写内容需和您提供的文件完全一致

举例：加拿大

填写管理员信息登记

填写管理员基础信息，并通过微信扫码进行绑定

管理员姓名的填写方式需要和您提供的证件一致

注意管理员手机区号，若是海外，请重选海外区号，否则收不到验证码。

三、微信小程序认证

开始前，您需要准备好以下材料：

1.《企业注册证》或者《商业许可证书》：原件照片或扫描，或者复印件加盖企业公章后扫描；保存为图片格式

2. 联系人正反面身份证件彩色扫描件； 3. 申请公函（中文简体认证公函、中文繁体认证公函、英文认证公函）； 4. 联系人手机电话对账单。若联系人手机电话对账单无法提供，可上传公司办公电话对账单（需加盖公章），或银行卡对账单； 5. 其他证明材料（若是选择商标命名的，请选择正确的商标注册书或者是商标授权书）。

如海外公司没有公章，需要加盖公章的位置需写上法人+和证件名字相同的签字

同意协议

填写企业资质信息

主体证件格式可参考：境外热门地区/国家常见主体证件示例

◽️上传《企业注册证》或《商业许可证》原件照片或扫描件

格式要求：原件照片、扫描件或者加盖公章的复印件，支持.jpg.jpeg.bmp.gif .png格式照片，大小不超过5M。

上传其他相关资料

这一步您需要准备的材料有：

◽️申请公函 境外主体账号认证公函示例

◽️运营授权书 境外主体账号运营授权书填写示例

◽️手机或银行对账单 （三个月）

◽️身份证件（正面）

◽️身份证件（反面）

*如果提供中国护照，姓名就是中文+拼音

-请注意，该步骤上传材料均不支持PDF版本，仅支持图片格式

-申请主体为公司全名

-您在运营授权书中需要填写的账号ID已展示在您的公函中 （gh_开头）

运营授权书：申请主体为公司全名。原始id查看申请公函内的id（gh_开头）。指定管理员姓名需按身份证填写（如果海外驾照是英文，就填英文）。申请主体授权代表请填写公司法人的姓名，正楷填写。海外公司无公章可以不用盖公章。

运营授权书的指引详见https://kf.qq.com/faq/220907BR7vUF220907N7Zv2q.html

填写认证联系人信息

联系人姓名填写方式需和提供的证件保持一致

如果是海外手机号码注意区号更改

支付费用

提交材料后，选择电子发票，支付认证费用

四、等待审核

如需补交资料，审核人员会在1-3个工作日内给你的预留邮箱邮件告知。按要求补交即可

邮件可以无限提交

微信后台更新信息限制最多3次

提交后如因提交的注册文件里没有显示出法人的姓名，邮件反馈或需要手持证件拍照

具体请以微信审核人员邮件反馈为准！

五、审核成功

认证成功后，您需要完成小程序基本信息和类目补充
