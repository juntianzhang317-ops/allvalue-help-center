# AllValue 帮助中心

基于飞书知识库生成的静态网站，帮助商家无需登录即可访问。

## 功能

- ✅ 完整还原知识库目录层级
- ✅ 全站搜索（标题搜索）
- ✅ 自动同步（每6小时）
- ✅ 响应式设计（手机/电脑）
- ✅ 无需登录即可访问
- ✅ 自定义域名支持
- ✅ 全球CDN加速（国内外均可访问）

## 自动同步

本仓库使用 GitHub Actions 每6小时自动同步飞书知识库内容。

## 技术栈

- **飞书 API** — 知识库内容获取
- **GitHub Actions** — 定时同步
- **Cloudflare Pages** — 网站托管与全球CDN
